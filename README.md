# Radhe Stonex — Premium Marble & Architectural Stone

Radhe Stonex is a high-fidelity, component-based React application designed to showcase a world-class collection of imported marble and architectural stones. The project emphasizes a "Modern Luxury" aesthetic, featuring razor-sharp textures, glassmorphism UI, and cinematic animations.

## 💎 Design Philosophy: "Obsidian & Gilt"
The application adheres to a strict design system defined by:
- **Tonal Depth**: Layered surfaces instead of borders.
- **Visual Clarity**: High-resolution assets with optimized rendering for maximum texture detail.
- **Cinematic Interactions**: Smooth transitions powered by Framer Motion.
- **Obsidian Theme**: Deep, dark surfaces accented with premium gold and marble highlights.

## 🚀 Tech Stack
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Icons**: [Material Symbols Outlined](https://fonts.google.com/icons)

## 🏗️ Project Structure
- **/src/pages**: Contains the core site sections (Home, About, Products, Gallery, Contact, ProductDetail).
- **/src/components**: Reusable UI components including the glassmorphism `Navbar` and the `MarbleScrim`.
- **/src/assets**: High-resolution marble textures and branding assets.

## 🛠️ Setup & Local Development
1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Run Dev Server**:
   ```bash
   npm run dev
   ```
3. **Build Package**:
   ```bash
   npm run build
   ```
